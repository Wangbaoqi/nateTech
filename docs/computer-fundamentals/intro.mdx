# 体系脑图

在计算机基础学科中，计算机基础主要有以下内容：

1. **计算机网络**
2. **计算机操作系统**
3. 计算机组成原理
4. **数据结构与算法**
5. 数据库
6. **选一门编程语言(比如C++, C, JavaScript)**
7. **编译原理**
8. 软件工程
9. **设计模式**

针对不同的领域的开发，可以选择不同的着重点，对于大部分的开发者，了解以及熟悉以上加粗模块，可以对计算机开发有一个
更深层次的理解。

而有了网络，则是打开新世界的大门，有很多新奇的东西是需要去探索的。
